# 需求文档

## 1. 应用概述

### 1.1 应用名称
纯画

### 1.2 应用简介
纯画是一款专注于设计、画画、海报约稿的跨平台接稿/约稿平台,主打无AI生成的纯洁创作环境。支持网页版、微信小程序版本,用户可选择成为画师或约稿人。

### 1.3 应用图标
简单的猫猫简笔画

---

## 2. 用户与使用场景

### 2.1 用户角色
- **画师**:提供绘画服务,需通过认证
- **约稿人**:发布约稿需求,寻找合适画师
- **管理员**:平台运营人员,负责审核与管理
- **AI助手**:系统自动角色,向新注册用户发送欢迎消息

### 2.2 核心使用场景
- 约稿人浏览画师作品并发起约稿
- 画师接受约稿并完成创作交付
- 用户在社区分享绘画心得与创作内容
- 海外用户切换语言使用平台
- 用户通过消息系统与画师/约稿人沟通

---

## 3. 页面结构与功能说明

### 3.1 整体页面结构

```
纯画
├── 首页/广场(未登录可访问)
│   └── 画师作品浏览、画师主页入口
├── 社区(需登录才能查看内容)
│   ├── 帖子列表
│   └── 帖子详情
├── 画师主页
│   └── 橱窗展示(作品列表)
├── 约稿流程
│   ├── 发起约稿页
│   ├── 对话/协商页
│   └── 支付页(定金/尾款)
├── 消息中心
│   ├── 左侧:消息列表
│   └── 右侧:对话窗口
├── 登录/注册页
│   ├── 微信授权登录
│   ├── 账号密码登录
│   └── 角色选择(约稿人/画师)
├── 画师认证流程页
│   ├── 步骤1:选择画师标签
│   ├── 步骤2:上传作品
│   └── 步骤3:等待审核提示
├── 个人中心
│   ├── 申请成为画师(约稿人角色)
│   ├── 查看认证状态(画师角色)
│   ├── 我的订单
│   └── 举报记录
└── 后台管理(平台运营人员)
    ├── 画师认证审核
    ├── 举报审核
    └── 管理员权限管理
```

### 3.2 首页/广场
- 展示平台画师作品,未登录用户可正常浏览
- 支持点击进入画师主页
- 无需登录即可访问全部内容
- 页面右上角提供语言切换按钮(中文/English)
- **作品卡片展示**:
  - 作品图片
  - 画师标签(如手绘-丙烯、板绘-厚涂等)
  - 画师名称
- **多语言支持**:切换至英文版时,作品卡片上的画师标签需显示对应英文翻译

### 3.3 社区
- **访问限制**:用户必须登录后才能查看社区内容;未登录用户访问社区时,跳转至登录页
- **帖子列表页**:展示所有用户发布的帖子,按发布时间倒序排列
- **帖子详情页**:展示帖子正文、图片(如有)、评论列表
- **发布帖子**:登录用户可发布帖子,支持文字内容与图片上传,适用于分享绘画心得、创作过程、交流讨论等
- **评论功能**:登录用户可在帖子下发表评论

### 3.4 画师主页(橱窗展示)
- 展示画师基本信息及认证状态
- 展示画师上传的作品及对应画师标签(如手绘-丙烯、板绘-厚涂等)
- 约稿人可在此页面发起约稿
- **多语言支持**:切换至英文版时,画师主页所有文本内容(包括画师标签)需显示对应英文翻译

### 3.5 登录/注册页
- **登录方式**:
  - 微信授权登录(小程序端)
  - 账号密码登录(网页端与小程序端均支持)
- **注册功能**:
  - 用户填写账号、密码、确认密码
  - **角色选择**:注册时必须选择成为约稿人或画师
    - 选择约稿人:注册成功后直接完成,自动登录
    - 选择画师:注册成功后跳转至画师认证流程页
- **管理员账号**:
  - 初始管理员账号需在后端数据库中预设
  - 管理员登录后可访问后台管理功能
- **注册后自动触发**:用户注册成功后,AI助手自动向该用户发送一条欢迎消息

### 3.6 画师认证流程页
- **触发场景**:
  - 注册时选择成为画师
  - 约稿人在个人中心点击申请成为画师
- **页面顶部进度条**:显示当前步骤(1/3、2/3、3/3)
- **步骤1:选择画师标签**
  - 标题:请选择您的创作类型(可多选)
  - 标签分类:
    - 手绘类:丙烯、水粉/水彩、铅笔、素描等
    - 板绘类:厚涂、平涂等
  - 用户可多选标签
  - 点击下一步进入步骤2
- **步骤2:上传作品**
  - 标题:上传您的画作(不少于三张)
  - 提供批量上传功能
  - 上传数量限制:至少3张
  - 点击下一步进入步骤3
- **步骤3:等待审核提示**
  - 标题:感谢您选择成为画师
  - 提示文案:这一定是一个勇敢的决定,请耐心等待审核
  - 提交后进入后台人工审核队列
  - 点击完成返回个人中心
- **审核状态查看**:画师可在个人中心查看认证状态(审核中/已认证/未通过)
- **审核通过后**:画作自动出现在画师主页和广场推送
- **审核未通过**:系统向画师反馈具体原因,画师可重新提交

### 3.7 约稿流程页面

**发起约稿页**
- 约稿人在画师主页点击发起约稿,进入需求填写页
- 填写内容:
  - 作品需求描述(文字)
  - 截止时间(可分别勾选草图截止时间和成图截止时间)
  - 价格(起价不低于70元)
  - 参考图片(可选上传)
- 提交后进入对话/协商页

**对话/协商页**
- 约稿人与画师双方实时聊天
- 设置「允许AI接稿」/「不允许AI接稿」选项,由约稿人在发起约稿时选择
- 画师可在对话中修改价格和截止时间,约稿人确认后生效
- 双方协商一致后,约稿人发起定金支付(30元,通过微信支付完成,由平台代管)
- 画师在截止时间内在对话页上传草图/成图
- 约稿人在对话页确认作品:
  - 不满意:可申请重画(不退定金)或退出约稿(退25元定金,寻找下一位画师)
  - 满意:确认收货,支付尾款(通过微信支付完成)
- 约稿人可在收到原图后24小时内提交举报,填写举报理由,等待人工审核

### 3.8 消息中心
- **页面布局**:
  - 左侧:消息列表区域
  - 右侧:对话窗口区域
- **左侧消息列表**:
  - 展示所有对话会话,按最新消息时间倒序排列
  - 每个会话显示:
    - 对方头像
    - 对方昵称
    - 最新一条消息预览
    - 未读消息数量(如有)
  - 点击会话进入右侧对话窗口
- **右侧对话窗口**:
  - 顶部显示对方昵称
  - 中间区域展示历史消息记录,按时间顺序排列
  - 底部为消息输入框,支持文字输入与发送
- **AI助手消息**:
  - AI助手作为特殊会话出现在消息列表中
  - 用户注册成功后,AI助手自动发送欢迎消息:「感谢你使用纯画」
  - AI助手消息在对话窗口中正常展示

### 3.9 举报功能
- 入口:对话/协商页,收到原图后24小时内可操作
- 约稿人填写举报理由并提交
- 进入后台人工审核队列
- 审核通过后,退约稿人80%已付款项
- 审核结果通知举报人和被举报画师

### 3.10 个人中心
- 查看和编辑个人信息
- **约稿人角色**:
  - 查看约稿订单
  - 查看举报记录
  - 申请成为画师(入口按钮)
- **画师角色**:
  - 查看认证状态
  - 管理橱窗作品
  - 查看接稿订单

### 3.11 语言切换功能
- 所有页面右上角提供语言切换按钮
- 支持中文与英文切换
- 切换后页面所有文本内容自动更新为对应语言,包括:
  - 页面标题、按钮、提示文案
  - 广场作品卡片上的画师标签
  - 画师主页的画师标签
  - 所有用户界面元素
- 用户选择的语言偏好保存在本地,下次访问时自动应用

### 3.12 后台管理
- **画师认证审核**:查看待审核认证申请,审核通过或驳回(驳回需填写原因)
- **举报审核**:查看待审核举报,审核通过或驳回,审核通过后触发退款流程
- **管理员权限管理**:管理员可在后台查看和管理其他管理员账号
- **访问权限**:仅管理员账号可访问后台管理页面

## 4. 业务规则与逻辑

### 4.1 社区访问规则
- 社区页面(帖子列表、帖子详情、发布帖子)均需登录后方可访问
- 未登录用户点击社区入口,跳转至登录页,登录成功后返回社区
- 广场/首页无需登录即可访问

### 4.2 角色选择与认证规则
- 注册时必须选择成为约稿人或画师
- 选择画师后必须完成认证流程(选择标签 → 上传作品 → 等待审核)
- 约稿人可在个人中心申请成为画师,申请后进入相同的认证流程
- 画师认证通过后,作品自动出现在广场和画师主页
- 画师标签在广场和画师主页展示时,需根据当前语言显示对应翻译

### 4.3 画师标签分类
- **手绘类**:丙烯、水粉/水彩、铅笔、素描等
- **板绘类**:厚涂、平涂等
- 画师可多选标签
- 标签在中英文环境下需提供对应翻译

### 4.4 价格规则
- 约稿起价不低于70元
- 定金固定为30元
- 尾款 = 最终协商总价 - 30元定金

### 4.5 退款规则
| 场景 | 退款金额 |
|------|----------|
| 约稿人申请重画 | 不退定金 |
| 约稿人退出约稿,寻找下一位画师 | 退25元定金 |
| 举报审核通过 | 退80%已付款项 |

### 4.6 举报时限
- 约稿人收到原图后24小时内可提交举报,超时不可举报

### 4.7 定金代管规则
- 约稿人支付定金后,资金由平台代管,不直接打给画师
- 双方确认收货后,平台将款项结算给画师

### 4.8 AI接稿规则
- 约稿人在发起约稿时选择是否允许AI接稿
- 若选择「不允许AI接稿」,画师承诺不使用AI生成内容
- 约稿人可在收到原图后24小时内提交AI举报

### 4.9 管理员权限规则
- 初始管理员账号需在后端数据库中预设
- 管理员登录后可访问后台管理所有功能
- 普通用户无法访问后台管理页面

### 4.10 语言切换规则
- 用户选择的语言偏好存储在浏览器本地存储或用户账号设置中
- 页面刷新或重新访问时自动应用用户上次选择的语言
- 未登录用户的语言偏好仅保存在本地存储中
- 画师标签需提供中英文对照翻译表,切换语言时自动匹配显示

### 4.11 AI助手消息规则
- 用户注册成功后,系统自动触发AI助手发送欢迎消息
- 欢迎消息内容:「感谢你使用纯画」
- AI助手消息在用户的消息中心中展示,作为独立会话存在
- AI助手仅发送系统预设消息,不支持用户主动向AI助手发送消息

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 未登录用户访问社区 | 跳转登录页,登录后返回社区 |
| 画师认证上传作品少于3张 | 提示至少上传3张作品,禁止进入下一步 |
| 画师认证审核未通过 | 系统通知画师并反馈具体原因,画师可重新提交 |
| 约稿人超过24小时后提交举报 | 系统提示举报时限已过,不受理 |
| 支付失败 | 提示支付失败,订单保留,可重新发起支付 |
| 画师超截止时间未上传作品 | 约稿人可申请退款,退还已付定金 |
| 举报审核不通过 | 通知举报人审核结果,款项不退 |
| 非管理员尝试访问后台 | 提示权限不足,跳转至首页 |
| 英文版广场画师标签未翻译 | 检查标签翻译映射表,确保所有标签有对应英文翻译 |
| 英文版画师主页未翻译 | 检查多语言配置文件,确保画师主页所有文本元素有英文翻译 |
| AI助手消息发送失败 | 系统记录失败日志,不影响用户注册流程 |
| 用户删除AI助手会话 | 允许用户删除,不影响系统功能 |
